<h1>Requirements</h1>

```
git
figlet
curl
date
```

<h1>Installation</h1>

```
git clone https://github.com/The-Amoghavarsha/WOTH.git
cd WOTH
chmod +x woth

*Or use alt-woth for better results* 
chmod +x alt-woth
```

<h1>Run</h1>

```
./woth
<h3> Or use alt-woth for better results </h3>
./alt-woth
```

<h1>Options</h1>

```
There is only one option to get the words of other days. 

./woth -o 
```
